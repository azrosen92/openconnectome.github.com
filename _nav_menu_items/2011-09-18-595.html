In alignment with our open science objectives, all the code for this project is (and will continue) to be developed open source, and hosted at our github site: openconnectome. Service Description Data hosting Our Bruster (brain-cluster) is large enough to store nearly any modern connectome data set. If you would like to make your data available to others for any purpose, including gaining access to state-of-the-art analysis and machine vision pipelines, please simply contact us, and we will get the ball rolling. Web Viewing Collaborative Annotation Toolkit for Massive Amounts of Image Data (CATMAID) is designed to navigate, share and collaboratively annotate massive image data sets of biological specimens. The interface is inspired by Google Maps, enhanced to allow the exploration of 3D image data. Volume Cutout Service Our RESTful API enables you to select any arbitrary volume of our 3d database (3ddb), and receive a link to download in HDF5 format, designed specifically for matlab, C, C++, C#, and python (with support for other programming languages available upon request). Volume Viewer A web & GPU enabled stand-alone app for viewing volumes at arbitrary cutting planes and zoom levels.  This app seamlessly interfaces directly with our 3ddb. Coming soon…. Machine Vision Pipeline We are building a machine vision pipeline that pulls volumes from our 3ddb and outputs neural circuits.  Of course, this is work in progress.  As soon as we have a stable version, we will release it here. Mr. Cap Magnetic Resonance Connectome Automated Pipeline. Our pipeline built on JIST/MIPAV for high-throughput estimation of connectomes from diffusion and structural imaging data.
